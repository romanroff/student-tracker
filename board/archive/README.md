# Milestone Archive

Move completed milestone files here after every task in the milestone is done
and accepted.

Archive the full `milestone*.md` file. Do not split completed tasks into a
shared archive document.

After moving a milestone:

- update `BOARD.md` so the active index no longer points to the old location
  unless the archived milestone should remain visible as history
- preserve task ids, owners, final statuses, changed files, validation results,
  blockers, and follow-up notes inside the moved milestone file
